## Information

### Keybinds

- WASD Keys: Move Lin
- Arrow Keys: Move Puro
- 1-4 Keys: Transfurs
- E Key: Lin Punch
- Q Key: Lin Mourn
- Z Key: Puro Read
- X Key: Ming Cat Stand
- M Key: Mute/Unmute
- N Key: Next Song
